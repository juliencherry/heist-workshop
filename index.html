Introductory Website Design Workshop

About

Welcome! My name is Julien Cherry and I’m offering an introductory website design workshop for HEIST participants on March 22nd, 2019.

Purpose of this Workshop

We’re going to learn the basics of HTML and CSS to create a personal website, emphasizing good website design habits. We’ll end up with a simple website hosted on GitHub Pages.

Agenda

  • Introduction and setup
    • GitHub account and GitHub Desktop
    • Atom with atom-live-server package
  • Creating and cloning GitHub Pages repo
  • Interactive introduction to HTML and CSS
  • Continuing making personal websites (class-directed)
  • Pushing websites to GitHub Pages
  • How to use browser tools

Resources

  • Mozilla HTML elements reference - An excellent guide to all the HTML elements
  • Mozilla CSS reference - Same thing but for CSS properties and more
  • Better Motherfucking Website - A tongue-in-cheek website encouraging lean HTML and CSS—also a good starting point for a simple stylesheet
